# Ontologie de la plateforme Les Entreprises s'Engagent

Ce document veut constituer une référence partagée entre métier et technique pour la modélisation conceptuelle de la plateforme.

## Préambule

La plateforme *Les Entreprises s'Engagent* accompagne les entreprises françaises dans leurs démarches d'engagement sociétal. Au fil du temps, le vocabulaire utilisé dans le code, les interfaces et les échanges métier a divergé, créant des zones d'ambiguïté qui freinent l'évolution du produit.

Ce chantier ontologique vise à établir un **langage commun** entre toutes les parties prenantes : équipe produit, développement, animation territoriale et direction. L'objectif est de clarifier les concepts fondamentaux qui structurent notre activité.
